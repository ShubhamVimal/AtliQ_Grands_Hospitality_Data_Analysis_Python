# Hospitality Data Analysis - AtliQ Grands

## Overview
This project analyzes hotel booking data to understand patterns in occupancy, revenue, and customer behavior.

The analysis is done using Python in Jupyter Notebook with a structured, question-based approach.

---

## Problem Context
AtliQ Grands is a chain of five-star hotels operating across India.

The company has been facing a decline in market share and revenue. This project explores historical data to understand patterns and identify areas for improvement.

---

## What This Project Covers
- Data cleaning and preparation
- Handling missing values and outliers
- Data transformation
- Question-based analysis
- Insight generation

---

## Key Insights
- Occupancy varies across room categories and cities
- Some cities consistently perform better
- Weekend and weekday booking patterns differ
- Revenue shows clear monthly trends
- Booking platforms contribute differently to revenue
